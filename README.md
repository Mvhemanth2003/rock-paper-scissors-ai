🧠 Rock Paper Scissors AI (Computer Vision Project)

This project is a real-time hand gesture classifier that detects Rock, Paper, or Scissors using a webcam.

🚀 Features
Real-time webcam prediction
Built using Deep Learning (ResNet50)
Transfer Learning from ImageNet
Image preprocessing using OpenCV
Live prediction display
🛠️ Tech Stack
Python
TensorFlow / Keras
OpenCV
NumPy
📂 How it works
Capture image from webcam
Preprocess using ResNet50 preprocessing
Pass through trained model
Predict gesture (rock/paper/scissors)
▶️ Run locally
pip install -r requirements.txt
python main.py
📸 Output
Displays prediction on screen in real-time
📌 Note

Model file is not included due to size. You can train your own using the dataset.
